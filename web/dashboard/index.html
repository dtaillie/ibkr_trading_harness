Help

Local-first framework, not a strategy recommendation
Start Here
  1. Confirm Gateway/API status in Overview.
  2. Open Data Library and verify your real data roots are visible.
  3. Inspect a saved dataset before using it in a replay.
  4. Build or load a draft in Workbench.
  5. Run replay or simulated paper, then inspect Performance and Runs.
Why only a few symbols?

The catalog only scans configured data roots and returns the selected limit. If you see only SPY and QQQ, the server is likely pointed at example data. Add your cache/history directories with --data-root or private local config.

Glossary
Replay
Run a plugin against saved bars.
Shadow
Observe decisions without submitting orders.
Simulated paper
Use local simulated fills and account state.
Paper
Submit to a broker paper account.
Artifact
Saved summary, decisions, orders, fills, and account snapshots.
Stale bar
Latest market data is older than expected.
Safe Boundaries

Public examples are intentionally no-edge. Private strategies should live in ignored config/plugin files. The dashboard summarizes decisions and artifacts without returning raw private signal payloads.
